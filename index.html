Anyone can learn to code

We start from the basics, so you don't need any coding experience – you just need to be ready to try new things. Any girl aged 15 or over can sign up.

Even if you plan on a different career, learning to code is an awesome skill. Business owner? Engineer? Designer? Architect? Coding gives you superpowers in these fields too.

What happens at Girl Code?

You will work as a team to build a web application – an interactive website similar to Tumblr, Pinterest, or Twitter.

In the first 3 sessions, we teach you the foundation skills. After that, you will take the lead in building a web application.

You'll come across the same challenges you'd face at a real tech company. But you won't be facing them alone! Our mentors will help you, and you'll be working with a team of your peers.

The tools we teach at Girl Code are the same tools used by top employers like TradeMe, Vend and Orion Health.

By the end of the course, you will know how to:

  • Create a website using HTML and CSS
  • Add interaction to a site using JavaScript
  • Let your users chat and post content through Node.js
  • Work together on the same code using Git
  • Use Agile planning techniques to stay on track and make tough decisions
  • Release your project into the world!

Who runs Girl Code?

Alice and Matthew are helping to build a more diverse and interesting tech community in Auckland.

We've run coding workshops for Auckland Libraries, FutureInTech, schools and other organisations. Over the years, we noticed that most people teach coding as a solo activity, which doesn't reflect the team environment you find in the workforce.

Girl Code is designed to show what it's really like to work on software with other people. Although team projects are more challenging, we keep a 1-to-5 teacher-student ratio to make sure you can always get help when you need it.
