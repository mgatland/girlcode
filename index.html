Girl Code

An eight-week software adventure for girls aged 15-19


Learn More!

Do you like the internet?


Do you like making things?


Girl Code is an 8-week course where you work in a team to design, create and publish a web app.

We start from the basics, so you don't need any coding experience â€“ you just need to be excited about learning new things!

Any girl aged 15 to 19 can join in.

What happens at Girl Code?

Ten students will meet each Thursday and work on a software project.

For the first few weeks, our mentors will teach you the skills you need. After that, you'll be running the project, setting goals and deciding what to make next.

You'll develop skills in

  • Software design
  • Leadership and team work
  • Planning and project management
  • User experience design
  • Coding tools like HTML, CSS, JavaScript, Node and Git

When & Where

Girl Code runs for 8 weeks, from 30th July to 17th September.

We meet on Thursday nights, from 6:30pm to 8:30pm at GridAKL in Wynyard Quarter, downtown Auckland.

The cost is $400 for the 8 week course.

Who runs Girl Code?

Alice

Alice is passionate about helping women succeed in STEM. She's taught at and attended loads of coding workshops, and knows what works and what's still being figured out.

Years of improv comedy training make Alice an expert at getting groups of people collaborating creatively.

Matthew

A software engineer and game designer, Matthew has taught hundreds of young people to code with OMG Tech! and Auckland Libraries.

Alice and Matthew both work on Pop-Up Arcade, an inclusive space to play NZ-made video games.
