var getRandomConcept = function () {
  var people = [];
  people.push("time traveller");
  people.push("school teacher");

  var jobs = [];
  jobs.push("runs a hotel");
  jobs.push("solves mysteries");

  var myConcept = "A " + pickRandom(people) + " who " + pickRandom(jobs);
  return myConcept;
};

Part 5: Emphasis on Adventure

Make the characters and the city appear italic, but the other words not appear italic. For example:

A school teacher and a cat who solves mysteries in London.

Clues:

You will need to add some HTML code inside the JavaScript.

JavaScript does not speak HTML. They are different languages. In JS, symbols like < and > are math symbols that mean less than and greater than.

However, if we put them inside speech marks "like this", that tells JS to ignore their special math meanings, and just use them literally how they look. This lets us insert them into the HTML page.


  //this works
  var example1 = "I saw a <i>hungry</i> cat";

  //this works:
  var catColor = "tabby";
  var example2 = "I saw a <i>" + catColor + "</i> cat";

  //this will not work, because the math makes no sense.
  //um, plus less than i? what are you talking about?
  var example2 = "I saw a " + <i> + catColor + </i> + "cat";

  //this will not work. What happens to the plus?
  var example3 = "I saw a +" catColor;
