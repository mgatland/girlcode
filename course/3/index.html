Working together

We make changes as we work.

Once we have a change we want to keep, we "commit" it.

This wraps up our change so can become part of the project.

Working together

The change is only on our computer.

To share it, we 'sync'.

This sends our commits to github.com, and fetches everyone else's commits.

Inside a commit

A commits is a set of changes

Most changes involve adding and deleting.

Here, we deleted the old text and added some new text.

That's how git works!
