Changing Colours Colors

We can change the color of anything using CSS.

The CSS is marked by a start tag like this: <style> and and end tag like this: </style>. The code between those tags is the CSS.

Once you've found the CSS, you'll see the words 'red' and 'blue'. Try changing them to other colors, save your page and see what happens.

When we're programming, we have to spell in US English, not NZ English. That means we spell 'color' without the 'u'.

Luckily, most words are still spelt the same way.

Did you change the colors? Remember to save and refresh.

Your CSS code might look like this now:

body {
  color: purple;
  background-color: orange;
}

Change the heading

Let's add a new CSS rule to make the page heading look different.

  • Find the line that says </style>
  • Add a blank line above it.
  • Add this code there:
h1 {
  text-align: center;
  font-size: 90px;
  color: white;
}

Save your changes, switch tabs to your page and check what happened.

You should see that the heading has changed.

Editing your page

The "body" of the page controls what words appear on your website.

The body starts with a <body> tag and ends with a </body> tag.

Everything between those two tags is inside the body.

Your page body probably looks like this:

<body>
    <h1>Welcome to my page</h1>
    <p>This is a paragraph.</p>
</body>

h1 is short for 'heading 1'. It represents the biggest heading on your page.

p is short for 'paragraph'.

Add another paragraph by adding a new line inside the body. Start the new line with <p> and end it with </p>. You can write anything you like in between the tags.

Remember: End tags have a slash before their name, like this: </body>

Adding an Image

Try adding this somewhere inside your page body:

<img src="http://mgatland.neocities.org/countryside.jpg">

An <img> tag shows an image. You can change the image by putting in a different link.

Side note: <img> tags are unusual because they don't have a closing tag. There's no such thing as </img>.

Links

Try adding this:

<a href="http://www.google.com">Here is a link.</a>

It should look like this:

Here is a link.

Pay close attention to where the pointy brackets go! The href part is inside the first tag.

Make your link go to a website you like to visit.

About HTML

The stuff with pointy brackets is HTML. Using HTML, you can change what the page says, and add images, but you can't change colors or add borders.

HTML is for what the page says, but not how it should look.

About CSS

A CSS rule tells your browser how to draw your HTML.

Here's an example of a CSS rule:

img {
  border: 10px dashed green;
  margin: 30px;
}

The green part says what the rule is for. This rule is for when you draw an img tag.

The curly brackets { and } mark the start and end of the instructions.

Each instruction is like this:

border: 10px dashed green;

There are hundreds of different rules you can use! They always have a : in the middle and a ; at the end. This one says to draw a dashed green border around the edge.

So the whole thing says: When you draw an img tag (an image), you must draw a border that is 10 pixels thick, green and dashed, and you must leave a margin of empty space around it that is 30 pixels wide.

Add the CSS to your page and see if it works!

Remember that CSS goes in the <style> section of the page.

If it has curly brackets {}, it's CSS. If it has pointy brackets <> it's HTML.

Basic layout

Some people have wide screens, but it's hard to read text that is too wide.

Let's put the contents of our webpage inside a container that won't get too wide.

Add this tag just before your <h1> tag:

<main>

Add this tag just after your last paragraph or image tag:

</main>

Now all your headings, paragraphs and images should be inside this new element. Finally, add a CSS rule to your <style> section to make the main element narrow:

main {
  max-width: 800px;
  margin: auto;
  background-color: lightblue;
}

Here are some more techniques that might help:

make an image centered

CSS:

img {
  display: block;
  margin-left: auto;
  margin-right: auto;
}

This means an image will be centered on its own line, and nothing is allowed to appear beside it.

Is your image too wide? If it is, you can set its size by adding a CSS rule to set the width:

width: 80%;

Or try this one instead. Percentage (%) widths change based on the width of the parent. Pixel (px) widths do not:

width: 400px;

How Borders, Padding and Margins work

Imagine that html tags create a box. The box can have a border.

HTML:

<p>this has a border</p>

CSS:

p {
  border: 1px solid blue;
}

How it looks:

this has a border

The border is very tight on the left side. We can add padding, which is space between the border and the words inside.

Add this CSS between the curly brackets:

padding: 10px;

How it looks:

this has a border and padding.

Margin is space OUTSIDE the box.

Add this CSS between the curly brackets:

margin: 40px;

How it looks:

this has a border and padding AND margin – see how there is more space on each side of the box, and above and below it?

Right click on the box above, and choose Inspect. This will open the Web Developer Tools. It can be pretty complicated, but somewhere you should see this diagram which shows you how thick the borders, margins and padding are on:

If you hold your mouse over different parts of the diagram, they will be highlighted on the web page. This can help you figure out why things appear where they are!

You can also double-click on any of the numbers to change them. Try making the top border thicker.

The Developer Tools are just for trying ideas out. The changes you make there are not saved and will disappear when you refresh the page.

Clean up your CSS

You might now have two different CSS rules for img and two different rules for p. Keep your CSS easy to read by combining these rules together.

Messy example:

p {
  color: white;
}
img {
  padding: 9px;
}
p {
  margin: 10px;
}

Tidy it up by combining the two p sections:

p {
  color: white;
  margin: 10px;
}
img {
  padding: 9px;
}

Now all the rules for paragraphs are together. Nice and tidy!

Making one thing different from the others with CSS classes

What if you want one paragraph to be red, and another one to be blue?

You can give any HTML tag a 'class', which is like an extra label. You make up the label. Here I called it loud:

HTML:

<p>This is a normal paragraph.</p>
<p class="loud">This is a LOUD paragraph!.</p>

Now you can write a CSS rule that applies to anything that has that class.

CSS:

.loud {
  color: red;
  border: 2px solid orange;
  background-color: pink;
}

The result:

This is a normal paragraph.

This is a LOUD paragraph!

Note that in our CSS rule, there is a dot in front of the word "loud". That shows that it's not a tag name (like p or img), it's a class name.

Classes let you quickly add your styles to anything you want.

Putting things inside other things

We can create advanced layouts by putting some HTML elements inside other ones.

Here we have put a h4 heading and some 'option' divs inside a div we called a 'survey'.

A div is just a general thing - unlike p (for paragraph), div has no meaning on its own. We can use it to mean whatever we want.

HTML:

<div class="survey">
  <h4>How many cats is enough cats?</h4>
  <div class="option">one cat</div>
  <div class="option">two cats</div>
  <div class="option">ten cats</div>
  <div class="option">no cats</div>
</div>

CSS:

.survey {
  border: 2px solid orange;
  padding: 10px;
  margin: 10px;
  width: 340px;
  text-align: center;
}

.option {
  background-color: wheat;
  padding: 4px;
  margin: 4px;
  border-radius: 15px;
}

How it looks:

How many cats is enough cats?

one cat
two cats
ten cats
no cats

Some terms

Here are some words you can use to talk about HTML:

This whole thing is called an HTML element:


<h1>Welcome to my page</h1>

The element is made out of 3 parts:

  • the start tag: <h1>
  • the content: Welcome to my page
  • the end tag: </h1>

Some elements contain attributes:


<a class='loud' href='https://www.google.com'>My favourite website</a>

These parts are attributes:

  • class='loud'
  • href='https://www.google.com'

Attributes always appear inside the start tag.

Some special tags like <img> only have a start tag, no closing tag and no content!

So now you know: HTML is made out of elements, and elements are made out of a start tag, content, and an end tag. The start tag might have attributes inside it.

That's the end!

There is a lot more you can do with HTML and CSS, but these are the basics. Remember what you've learned:

  • The HTML code in the page <body> is what appears on the screen
  • The CSS code in the <style> section tells your browser how to draw things
  • You can add images <img src="…"> and links <a href="…">
  • CSS can add borders, padding and margins
  • Classes let us set up rules then add them to whatever parts of the page we like
